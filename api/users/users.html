/users¶

GET /users/(string: username)¶

Show user profile.

Parameters:
  • username (string) – the username for which to obtain information
Response JSON Object:
 
  • username (string) – the username of the obtained user
  • name (string) – real name of the obtained user
  • points (int) – total score of the obtained user
Status Codes:
  • 200 OK – when user record was found. Returns user information
  • 404 Not Found – when user record was not found or was deleted

Example request:

GET /users/testuser HTTP/1.1
Host: example.com
Accept: application/json
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/json

{
    "username": "testuser",
    "name": "Test User",
    "points": 42
}
